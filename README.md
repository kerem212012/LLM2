# LLM2 Text Analysis

A minimal text analysis project using Anthropic Claude via Python.

## Requirements

- Python 3.12+
- `API_KEY` environment variable set for Anthropic
- `MODEL` environment variable optionally set (default: `claude-sonnet-4-6`)

## Install

```bash
python -m pip install -U .
```

## Usage

This project is intended for direct local execution, not as a website or HTTP service.

Run the script directly for batch analysis:

```bash
python main.py
```
